# Harsha Naveen Sai Manchi - Portfolio

Welcome to my personal portfolio repository! This is a single-page modern portfolio website built to showcase my skills, projects, and achievements as an Aspiring Cloud Developer and AI Enthusiast.

## Live Preview
*(You can host this for free on platforms like GitHub Pages, Vercel, or Netlify)*

## Features
- **Modern UI/UX**: High-quality dark mode theme featuring glassmorphism elements, CSS animations, and subtle noise overlays.
- **Fully Responsive**: Adapts seamlessly from desktop monitors down to mobile devices.
- **Dynamic GitHub Integration**: Automatically fetches and displays the latest repositories straight from GitHub using the GitHub REST API.
- **Smooth Navigation**: One-page layout with smooth scrolling and intersection observers for engaging section reveal animations.

## Technologies Used
- **HTML5** for semantic structuring.
- **CSS3** for styling, flexbox/grid layouts, variables, keyframe animations, and responsive media queries.
- **Vanilla JavaScript (ES6)** for API interactions, form handling, and scroll effects.

## How to Run locally
1. Clone the repository:
   ```bash
   git clone https://github.com/harshanaveensaimanchi-del/MY-PORTFOLIO.git
   ```
2. Navigate into the directory:
   ```bash
   cd MY-PORTFOLIO
   ```
3. Open `portfolio.html` in any modern web browser or run it using a local development server (like VS Code Live Server). No build process required!
